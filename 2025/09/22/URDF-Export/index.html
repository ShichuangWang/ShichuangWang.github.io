使用SolidWorks导出单个零件的URDF文件

发表于 2025-09-22 更新于 2025-10-01
Export URDF from SolidWorks

1.软件准备

  • VS Code
  • Mujoco
  • SolidWorks

2.安装SolidWorks插件和Mujoco

首先安装SolidWorks to URDF Exporter插件，下载链接（需要科学上网）如下： Releases · ros/solidworks_urdf_exporter

直接下载.exe文件，注意版本，我使用的是SolidWorks 2022，使用的URDF Exporter插件版本为1.6.0，（SolidWorks 2022 无法使用1.6.1版本的URDF Exporter插件） 接着安装Mujoco，下载链接： Releases · google-deepmind/mujoco 下载压缩包解压后，在其下的bin文件夹下有一个simulate.exe，就是Mujoco

3.导出URDF文件

  1. 打开SolidWorks，并打开零件。
  2. 注意！！！插件只能从装配体（.sldasm格式）导出URDF文件，不能直接从零件（.sldprt）以及其他格式的文件导出URDF，如果是非装配体格式的零件，可以点击从零件/装配体制作装配体，然后制作装配体之后再进行URDF文件导出，如果是装配体（.sldasm格式）的文件直接跳到第5步。
  3. 点击确定
  4. 开始装配体，这步完成之后就可以开始导出URDF文件了
  5. 首先需要建立坐标系，对于单个零件坐标系的选取目前我还没有找到特别的需要注意的地方，我一般是在质心处建立坐标系。依次点击工具栏：“装配体”–>“参考几何体”–>“质心”创建质心。
  6. 左侧面板选中质心，依次点击工具栏：“装配体”–>“参考几何体”–>“坐标系”创建坐标系
  7. 自定义三个轴的朝向或者使用默认，然后创建坐标系
  8. 点击菜单栏，“工具”–>“Tools”–>“Export as URDF”
  9. 自定义link的名字或者使用默认（我这里使用默认的，就叫base_link），接着选择坐标系，然后点击preview and export
  10. 只有一个零件装配体，直接点击next
  11. 选中base_link，然后点击Export URDF and Meshes
  12. 选择一个存储路径，存储导出的文件，并且一定要用英文命名

4.使用VS Code查看URDF文件

  1. 找到刚刚导出的URDF文件存储的位置，右键使用VS Code打开
  2. 安装 URDF Visualizer扩展
  3. 查看urdf文件
  4. 修改STL文件路径（我这里使用的是相对路径，也可以使用绝对路径），并点击右上角的preview预览

5.使用SolidWorks导出质量、惯量属性

  1. 为装配体各各部件选择对应的材料
  2. 材料选择界面有很多已有的材料，大致选一个相近的就行
  3. 选择好之后点击应用
  4. 配置导出质量参数的选选项
  5. 选择坐标系，重算质量参数与惯性张量（惯性张量选择第二个，由重心决定，对齐输出坐标系）
  6. 根据导出的质量以及惯性张量修改urdf文件，注意导出的urdf文件中惯性张量只有6项，对照着SolidWorks导出的参数进行修改就行，不需要增加SolidWorks多出来那3项，因为Solidworks导出的那个3×3的惯性张量矩阵是关于对角线对称的
